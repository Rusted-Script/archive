# Math in Rusted Script
You can use standard math symobls to complete equations in Rusted Script, such as;
- Plus(+)
- Minus(-)
- Multiply(*)
- Power(^)
- and Divide(/)


## Usage
Usage of math in *Rusted Script*; 
```
var x = 4 + 2 - 1 * 3 / 2 ^ 3
# also;
if x == 3 + 3 then 123 else 32
```
